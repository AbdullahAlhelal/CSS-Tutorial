Image Centering Examples

Method 1: Center with margin: auto

Convert the `< img >` to block and give it left/right auto margins. The image will center itself within its container.

Used When:

  • Quick, minimal markup
  • Works in all browsers (IE9+)
  • No extra containers required

Method 2: Center with Flexbox

Wrap the image in a flex container and use justify-content: center to center it.

Used When:

  • More versatile if you need vertical centering too
  • Can center multiple items in a row
  • Cleaner when you already use flex for other layout
